实验原理：
前后端分离架构原理：前端基于 VUE2 框架构建页面，通过 HTTP 请求与后端 Spring Boot 服务进行数据交互，后端负责数据处理与数据库操作，前后端通过 JSON 格式传输数据，实现职责解耦。
跨域访问原理：由于前后端运行在不同端口（前端默认 8080 端口，后端默认 8081 端口），浏览器存在同源策略限制，需通过后端配置 CORS（跨域资源共享）规则，允许前端指定域名的请求访问，实现跨域数据交互。
数据交互原理：前端通过 Axios 发送 GET 请求获取后端数据，通过 POST 请求提交修改后的数据；后端通过 Spring Boot 的 Controller 层接收请求，调用 Service 层处理业务逻辑，再通过 MyBatis 操作 MySQL 数据库，最终将结果返回给前端。
Element-UI 组件原理：借助 Element-UI 提供的 Table（表格）、Dialog（对话框）、Form（表单）等封装组件，快速构建前端界面，通过组件的 prop 属性绑定数据，事件绑定实现交互逻辑，简化界面开发流程。
